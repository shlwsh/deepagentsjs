---
description: 自动加载最近一次的会话内容到本次会话中
---

这个工作流会自动完成以下操作:
1. 扫描 `sessions/` 目录，找到时间戳最近的 `.json` 会话文件
2. 读取该文件中的原始历史消息
3. 通过后端 API 的 `/session/import` 接口将其导入为一个新的会话
4. 返回导入后的会话 ID 供用户继续对话

## 使用方法

```bash
/loadsession
```

## 工作流程

// turbo
1. **执行加载脚本** - 执行 `bun run loadsession`

## 预期效果

执行成功后，系统会返回一个新的 `restored_时间戳` 格式的会话 ID，该会话已包含您上次保存的所有历史消息。
